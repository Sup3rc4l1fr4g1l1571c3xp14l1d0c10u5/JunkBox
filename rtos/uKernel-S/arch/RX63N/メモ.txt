相変わらずのルネサス苦オリティで死た。

肝心のRXマイコンもRISCなんだけどARM（というかMIPS）の後追いっぽくて…

こんなんだからARMに負けるんだよ…。

以下設定方法
１．RXシミュレータのプロパティを開く
２．周辺機能シミュレーションの周辺機能シミュレーションモジュールを展開
３．CMT（コンペアマッチタイマ）とICU（割り込み制御ユニット）それぞれの設定を「使用する」に

■インラインアセンブラ仕様変更
M16Cでは asm("nop") などが使えたが、RXマイコンではＣコードとアセンブラの混在が不可能となった。
代わりに以下のようにする
------

#pragma inline_asm foo_inline_asm
static void foo_inline_asm(void) {
	アセンブラを直接記述
}

void foo(void) {
	... // 何らかの処理
	foo_inline_asm();	// インラインアセンブラ用のメソッドを呼び出す
	... // 何らかの処理
}

------


■コンテキスト仕様
コンテキストはR0がスタックポインタ
R1〜R4が引数や戻値に使う領域
R5〜R14が保存必須な領域

RTSはスタックから復帰先番地を取り出してPCに設定するだけだが、
RTEはスタックから復帰先番地を取り出してPCに設定してから、スタックからPSW(フラグ)を取り出してPSWに設定する
